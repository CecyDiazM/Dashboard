📊 Retail Analytics Dashboard
A mobile-responsive, data-driven web application designed to track, filter, and visualize key performance indicators (KPIs) and sales trends for retail operations. This dashboard transforms raw sales records into intuitive, interactive charts and structured summaries, enabling businesses to monitor their performance anytime, anywhere.

🚀 Key Features
Real-Time KPI Tracking: Instant summary metrics tracking Total Sales (Ventas Totales), Transaction Volume (Transacciones), Average Ticket Value (Ticket Promedio), and Total Products Sold (Productos Vendidos).

Categorized Breakdown: Dynamic visualizations (e.g., bar charts) illustrating sales figures categorized by department, such as Accessories, Food, Sports, Electronics, Home, Toys, and Clothing.

Timeline Evolution: An interactive line graph detailing historical sales fluctuations over time to monitor sales growth and seasonal trends.

Deep-Dive Data Table: An expandable detailed view section (Ver datos detallados) providing granular row-level data including date/time timestamps, exact store locations, and individual products sold.

Advanced Multi-Filter Sidebar: A sliding filter pane that allows quick categorization and slicing of data based on City, Product Category, and Payment Method (e.g., Bizum, Cash, Card, Bank Transfer).

🧠 Problems It Solves
Data Fragmentations: Consolidates multi-metric data (revenue, counts, dates, payment types) into a singular unified application interface, removing the need to manage scattered spreadsheets.

Lack of Mobility: Many legacy dashboard solutions require a desktop environment to render graphs properly. As demonstrated in chatbot_streamlit_phone.mp4, this application is fully responsive and optimized for mobile screens, giving store managers data access on the go.

Slow Insights Gathering: Finding specific data segments can be tedious. The multi-select filtering engine allows users to rapidly narrow down data—for instance, isolating "Card sales for Sports items in Barcelona"—re-calculating all dashboard KPIs and graphs instantly.

🛠️ Built With
Streamlit: For compiling clean, reactive UI components and data filters.

Plotly / Altair: Powering the interactive data graphs and timeline visualizations.

Pandas: Managing backend data manipulation, calculation of retail KPIs, and structural matrix filtering.
